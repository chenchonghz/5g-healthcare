三大运营商首批5G城市名单
6月6日，中国工业和信息化部向中国电信、中国移动、中国联通、中国广电发放5G牌照。这意味着，四家运营商将可以正式建设5G网络。

中国电信、中国移动、中国联通今年将分别在至少40个城市覆盖网络，并在部分城市推出5G服务。从国是直通车获得的信息看，中国移动与中国联通选择的40城一致，中国电信尚未披露40城市完整名单，但鹰潭、琼海是电信独有的首批5G城市。
